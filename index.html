Projetos em Ciência de Dados

Previsão de receita de vendas para uma rede de farmácias

Nesse projeto, foi utilizado Python, Estatística e técnicas supervisionadas de Machine Learning para fazer a previsão de receita de venda das próximas 6 semanas de uma rede de farmácias.

A ideia desse projeto é auxiliar o CFO na tomada de decisão, o qual precisa definir o budget de reforma das lojas da rede.

Para isso, foi criado uma solução que fornece a previsão de vendas individual de cada loja de forma automática e que pode ser consultada rapidamente através de um celular com um Bot no aplicativo do Telegram.

As ferramentas utilizadas foram:

  • Python, Pandas, Matplotlib e Seaborn
  • Jupyter Notebook para prototipar a solução
  • VSCode para implementação da solução
  • Algoritmos da biblioteca scikit-learnig
  • Streamlit Python framework web
  • Git e Github
  • Hospedagem em Nuvem Render

Identificação de imóveis para compra e venda a fim de maximizar lucro

Nesse projeto, foi utilizado a programação em Python para identificação de imóveis abaixo do preço médio de venda e definição do preço ideal de revenda a partir de uma análise exploratória de dados.

O resultado do projeto foi um dashboard com as visualização dos relatórios e Insiths gerados, através do framework Streamlit, o qual foi hospedado em um ambiente cloud e disponibilizado por um link web que pode ser acessado por qualquer dispositivo conectado à internet.

Caso fosse implementada, essa solução seria capaz de trazer um retorno de aproximadamente US$ 76 Milhões.
